## XiaoFeiGun

Windows下基于aria2的下载管理器, 易用免配置, 支持 HTTP/HTTPS、FTP、SFTP、磁力链接（Magnet）与 BT 种子下载。

### ✨主要特性
**下载任务管理**：支持添加、暂停、继续、删除任务，操作直观高效。  
**自动多线程加速**：利用 aria2 的多连接机制，自动提升下载速度。  
**多任务并行下载**：同时处理多个下载任务，互不影响。  
**自动更新 BT Tracker**：保持最新 tracker 列表，提高 BT/Magnet 任务连接效率。
